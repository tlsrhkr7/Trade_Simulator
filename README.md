# Crypto Trading Simulator

C로 작성된 Windows 콘솔 가상화폐 모의 거래 시뮬레이터입니다. 코인 가격이 무작위로 변동하며, 사용자는 매수·매도로 현금과 총 자산, 수익률을 관리합니다.

## 개요

5종의 가상 코인 가격이 매 갱신마다 무작위로 변동하는 환경에서 매수·매도를 반복하며 자산을 운용하는 콘솔 기반 모의 거래 프로그램입니다.

## 주요 기능

- 5종 코인(SDJ, LSO, PDH, ADH, JIH), 갱신마다 -100~+100 무작위 변동
- 실시간 총 자산·보유 현금·시세 테이블(현재가, 전일 대비 변동률·변동액)
- 매수/매도, 평균 매수가·수익률 계산
- 상승/하락 색상 출력, 가격 0 근접 시 상장폐지

## 동작 방식

```
[초기 자산 입력] → 반복:
  화면 초기화 → 가격 무작위 갱신 → 시세/포트폴리오 출력
  → 1)매수 2)매도 3)종료 선택 → 1초 대기
```

## 기술 스택

C · Windows (windows.h, system("cls"))

## 빌드 방법

```bash
# Visual Studio: 콘솔 앱(C) 프로젝트에 코드 추가 후 빌드
# 또는 MinGW
gcc main.c -o trading_sim.exe
./trading_sim.exe
```
